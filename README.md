# Tic_Tac_Toe

Marvel Edition Tic Tac Toe
A fun and interactive Marvel-themed Tic Tac Toe game where players battle as Iron Man and Thor in an exciting superhero showdown.

🚀 Features
🎮 Classic Tic Tac Toe gameplay
⚡ Marvel-inspired UI
🤖 Iron Man vs 🔨 Thor theme
🏆 Winner detection system
🔄 Restart game functionality
📱 Responsive design


✨ Smooth and interactive experience

Open the project folder
cd marvel-tic-tac-toe
Run index.html in your browser

🎯 Game Rules


Two players take turns


One player is Iron Man


Other player is Thor


First player to align 3 symbols wins


If all boxes fill without a winner, the game ends in a draw
